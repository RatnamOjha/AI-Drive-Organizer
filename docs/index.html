How it works

From chaos to
clarity in minutes

Three steps. Fully local. Works with every major file format your Drive holds.

🔍
01

Scan & Extract

Connects to your Drive via OAuth2. Extracts text from PDFs, DOCX, XLSX, Google Docs, Sheets, and even scanned images via Tesseract OCR — all in parallel, in batches.

🧠
02

Classify Locally

A TF-IDF weighted keyword classifier scores each file against 9 categories using 100+ curated terms. No data leaves your machine — confidence scores let you tune sensitivity.

📁
03

Organize & Report

Files above your confidence threshold are moved to clean category folders. A generated HTML + JSON report gives you a full audit trail of every decision made.

A CLI built
for power users

Every behavior is tunable. Run a safe dry-run first, then let it rip.

🔍

Dry-run mode

Simulate the entire run without touching a single file. See exactly what would move and where before committing.

🎯

Confidence threshold

Set a minimum score (0–1) — files below it land in Miscellaneous instead of being force-categorized.

⚡

Async batch processing

Files are processed concurrently in configurable batches — 10x faster than sequential on large drives.

📊

HTML + JSON reports

Every run generates a full audit trail: what moved, what was skipped, confidence scores, and reasons.

Privacy first

Your files never
leave your machine.

In a world of SaaS tools that vacuum up your data, Drive Organizer AI is different by design.

🔐

Zero external transfer

All NLP and classification runs entirely on your local CPU. No file contents are ever sent to a third-party server.

🗝️

Local token storage

Your OAuth2 token is stored in token.pickle on your device only — never uploaded anywhere.

📜

Fully open source

Every line of code is on GitHub. Audit exactly what runs — no compiled binaries, no hidden network calls.

🛡️

Minimal permissions

Requests only the Drive scope needed to read and move files. No Gmail, no Calendar, no Contacts access.

Quick start

Up and running
in four steps

From a fresh clone to organized Drive in under five minutes.

1

Create GCP Project

Go to Google Cloud Console, create a project, enable Drive API, and download an OAuth2 Client ID JSON as credentials.json.

2

Clone & Install

git clone the repo, create a venv, then run pip install -r requirements.txt. Tesseract must be installed for OCR.

3

Configure

Copy config.example.yaml to config.yaml. Adjust categories, confidence threshold, and batch size.

4

Run it

Start with --dry-run to preview. Then run without it to organize. Your browser opens once for Google login.
